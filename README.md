# Antarman-AI

Antarman AI — Local mental health AI for Indian languages. Runs on 16GB RAM using LLaMA 3.2 3B (LoRA), RAG over clinical sources (DSM-5, NIMHANS), real-time crisis detection (DistilBERT + MentalBERT).

## Features

- **Local & Private**: Runs entirely on your machine
- **Indian Languages**: Support for Hindi, Tamil, Telugu, Kannada, and more
- **Lightweight**: Optimized for 16GB RAM systems
- **Clinical Knowledge**: RAG integration with DSM-5 and NIMHANS resources
- **Crisis Detection**: Real-time mental health crisis identification
