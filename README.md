# Antarman-AI

Antarman AI — Local mental health AI for Indian languages. Runs on 16GB RAM using LLaMA 3.2 3B (LoRA), RAG over clinical sources (DSM-5, NIMHANS), real-time crisis detection (DistilBERT + MentalBERT).

## Overview

Antarman AI is a local mental health artificial intelligence system specifically designed for Indian languages. The system leverages advanced machine learning models and clinical knowledge bases to provide culturally relevant mental health support while maintaining complete privacy through local processing.

## Features

### Local & Private
- Runs entirely on your machine without requiring cloud connectivity
- All data processing happens locally, ensuring complete privacy and data protection

### Indian Languages
- Support for Hindi, Tamil, Telugu, Kannada, and more
- Culturally tailored responses for Indian language speakers

### Lightweight
- Optimized for 16GB RAM systems
- Built with LLaMA 3.2 3B model enhanced with LoRA (Low-Rank Adaptation)
- Efficient design for accessible deployment on consumer-grade hardware

### Clinical Knowledge
- RAG (Retrieval-Augmented Generation) integration with clinical sources
- Access to DSM-5 diagnostic criteria
- Integration with NIMHANS (National Institute of Mental Health and Neurosciences) resources
- Evidence-based responses grounded in clinical knowledge

### Crisis Detection
- Real-time mental health crisis identification
- Uses DistilBERT and MentalBERT models for crisis assessment
- Enables immediate detection of high-risk situations
